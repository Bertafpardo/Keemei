Keemei is an add-on for validating bioinformatics metadata stored in Google Sheets.

Keemei currently supports QIIME's metadata mapping file format (as defined here). Support will be added in the future for additional metadata file formats used in bioinformatics (e.g., formats defined by Qiita, SourceTracker, and MG-RAST).

For more information about Keemei, browse the docs.

Keemei was developed by Jai Ram Rideout (@jairideout) in the Caporaso Lab. See the full list of Keemei's contributors here.
